# Daily RSS

每日技术 RSS 聚合日报，基于 Andrej Karpathy 精选信源自动生成。

## 📁 目录结构

```
daily-rss/
├── README.md
├── 2026/
│   ├── 02/
│   │   ├── 2026-02-13.md  # 每日日报
│   │   └── ...
│   └── ...
└── archive/               # 归档目录
```

## 📊 日报格式

每份日报必须包含：
- 🔥 核心主题聚合
- 📈 各信源精选内容（**必须包含原文链接**）
- 📊 今日数据概览
- 💡 编者观察

## 🔄 自动化

日报通过 OpenClaw Cron 自动生成，每日早上 6:00 更新。
